### 🚀 MARK — AI Voice Assistant

**MARK** is a voice assistant designed to interact through both text and speech, helping users perform tasks efficiently with a clean, real-time interface.

---

### 🎤 Key Features

* **Voice Command Support**
  Speak commands using microphone input and get instant responses

* **Text-Based Chat System**
  Chat interface with real-time responses and typing indicators

* **Smart Command Execution**

  * Open websites (YouTube, Google)
  * Get time & weather updates
  * Tell jokes & handle basic queries

* **Real-Time Interaction UI**
  Smooth chat bubbles, live typing animation, and responsive design

* **Command Logs Tracking**
  Stores and displays previous commands with timestamps

* **Notes System**
  Save and manage personal notes directly from the assistant

* **Live News Fetching**
  Get latest headlines dynamically from backend API

* **Assistant Customization**
  Rename the assistant and personalize your experience

* **Quick Command Shortcuts**
  One-click buttons for frequently used actions

* **Backend Integration Ready**
  Works with a Python backend (`mark_backend.py`) for real-world data handling

---

### 🛠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask API)
* **APIs:** Speech recognition, news API, system commands

### 💡 Project Highlights

* Designed a **minimalist and modern UI** with smooth UX
* Implemented **voice + text hybrid interaction system**
* Built a **modular backend API structure** for scalability
